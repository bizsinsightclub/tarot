# 🎨 Pathfinder — 디자인 가이드

본 문서는 Pathfinder 앱(상황 기반 타로 결정 도구)의 시각 시스템을 정의한다.
딥 퍼플 기반의 미니멀·프리미엄 톤은 타로의 신비성·결정 도구의 권위감을 동시에 전달한다.

---

## 1. 컬러 팔레트

| 역할 | 컬러값 | 사용 위치 | Tailwind 토큰 |
|------|--------|----------|----------------|
| **배경** | `#2B2042` (딥 퍼플/네이비) | 전체 배경, 카드 뒷면 | `bg-pf-bg` |
| **주 텍스트** | `#FFFFFF` ~ `#F0EDF5` (화이트/오프화이트) | 본문, 헤드라인 | `text-pf-fg` |
| **액센트/라벨** | `#C8A2E8` ~ `#D4B8F0` (라벤더/라이트 퍼플) | 섹션 라벨 (Past/Present/Future), GO 톤 판정 | `text-pf-accent` |
| **구분선** | `#A07060` ~ `#C09080` (로즈골드/코퍼) | 헤더 하단 수평선, STOP 톤 판정 | `border-pf-rose` |
| **서브텍스트** | `#B0AABA` (라이트 그레이 퍼플) | 부가 안내, NEUTRAL 톤 | `text-pf-mute` |
| **경고 톤** | `#D89060` (옅은 오렌지) | GO_AND_REGRET 같은 경고 판정 | `text-pf-warn` |

> 💡 **디자인 의도**: 딥 퍼플 + 화이트/라벤더 조합은 신비·권위·미니멀을 동시에 표현하며, 결정 보조 도구가 가져야 할 "신뢰감"을 시각적으로 뒷받침한다.

## 2. 타이포그래피 하이어라키

### Level 0 — 헤더 / 브랜드 마크

| 항목 | 내용 |
|------|------|
| **텍스트** | `Pathfinder` (워드마크) |
| **스타일** | 산세리프 Light, 자간 넓게 (tracking ≈ 200) |
| **컬러** | 화이트 `#FFFFFF` |
| **크기** | 14–16pt (작게, 미니멀) |
| **정렬** | 중앙, 페이지 최상단 |

---

### Level 1 — 브랜드 슬로건 (Hero)

| 항목 | 내용 |
|------|------|
| **영문** | **FATE *or* CHOICE** |
| **한글** | "운명을 묻고, 선택을 내린다." |
| **서체** | 영문: **Didot / Bodoni 계열** 하이 콘트라스트 세리프 / 한글: 본명조 (또는 SM명조) |
| **스타일** | 영문 대문자, `or`만 **이탤릭 소문자** (시각적 변화·강조) |
| **크기** | 영문 약 **48–56pt**, 한글 약 **18–20pt** |
| **자간** | 영문 넓은 자간 (tracking +50~100) |
| **컬러** | 화이트 `#FFFFFF` |
| **정렬** | 중앙 정렬 |
| **컴포넌트** | `<BrandSlogan />` |

---

### Level 2 — 메인 타이틀

| 항목 | 내용 |
|------|------|
| **텍스트** | "결정이 필요한 순간" / "당신의 카드" |
| **서체** | 고딕 계열 **Bold** (Noto Sans KR Bold / 본고딕 Bold) |
| **크기** | 약 **24–28pt** |
| **웨이트** | **Bold (700)** |
| **컬러** | 화이트 `#FFFFFF` |
| **정렬** | 중앙 정렬 |
| **컴포넌트** | `<PageTitle />` (또는 `<h1>` 직접) |

---

### Level 3 — 섹션 라벨

| 항목 | 내용 |
|------|------|
| **텍스트** | "PAST" / "PRESENT" / "FUTURE" / "VERDICT" |
| **서체** | 고딕 계열 **Bold**, 영문 대문자 |
| **크기** | 약 **14–16pt** |
| **웨이트** | **Bold (700)** |
| **컬러** | 🟣 라벤더 `#C8A2E8` |
| **자간** | tracking +100 (간격 넉넉히) |
| **정렬** | 중앙 정렬 |
| **간격** | 카드 위 약 16–20px |
| **컴포넌트** | `<SectionLabel />` |

---

### Level 4 — 본문 / 카드 의미 캡션

| 항목 | 내용 |
|------|------|
| **텍스트** | 카드 한 줄 의미, 판정 설명 |
| **서체** | 고딕 계열 **Regular~Medium** |
| **크기** | 약 **14–16pt** |
| **웨이트** | Regular (400) ~ Medium (500) |
| **컬러** | 오프화이트 `#F0EDF5` |
| **정렬** | 중앙 정렬 |
| **행간** | 약 1.6~1.8 |
| **컴포넌트** | `<CardCaption />` |

---

### Level 4-CTA — 판정 배너 (Verdict Panel)

| 항목 | 내용 |
|------|------|
| **텍스트** | `STRONG GO` / `GO AND REGRET` / `STRONG STOP` 등 |
| **서체** | 고딕 계열 **Bold** + 영문 대문자 |
| **크기** | 약 **36–48pt** (페이지에서 가장 큰 단일 텍스트) |
| **웨이트** | **Bold (800)** |
| **컬러** | 톤별 가변: GO=라벤더, STOP=로즈골드, WARN=오렌지, WAIT=그레이퍼플 |
| **정렬** | 중앙 정렬 |
| **부가** | 한국어 부제 (예: "강하게 GO") 작은 크기로 하단에 |
| **컴포넌트** | `<VerdictPanel />` |

### Level 5 — 부가 안내 (Footnote)

## 3. 레이아웃 & 스페이싱

### 홈 화면

```
┌─────────────────────────────────────┐
│         [L0] Pathfinder             │  ← 상단 패딩 ~32px
├── 로즈골드 구분선 (1px) ────────────┤
│                                     │
│                                     │
│      [L1] FATE or CHOICE            │  ← 섹션 간 간격 ~80px
│        운명을 묻고, 선택을 내린다     │
│                                     │
│                                     │
│      [L3] 결정이 필요한 순간          │  ← 간격 ~64px
│                                     │
│      ┌──────────┐                   │
│      │ 살까 말까 │                   │  ← 큰 버튼 3개 세로 스택
│      └──────────┘                   │     (모바일) 또는 가로 (데스크탑)
│      ┌──────────┐                   │
│      │ 할까 말까 │                   │
│      └──────────┘                   │
│      ┌──────────┐                   │
│      │ 만날까 말까│                   │
│      └──────────┘                   │
│                                     │
└─────────────────────────────────────┘
```

### Reading 화면 (카드 + 판정)

```
┌─────────────────────────────────────┐
│         [L0] Pathfinder             │
├── 로즈골드 구분선 ──────────────────┤
│                                     │
│      [L2] 살까 말까                  │
│                                     │
│   PAST     PRESENT    FUTURE        │  ← L3 라벤더 라벨
│   ┌──┐     ┌──┐       ┌──┐          │
│   │  │     │  │       │  │          │  ← 카드 3장 가로
│   │  │     │  │       │  │          │     (모바일도 가로 유지, 폭 자동 축소)
│   └──┘     └──┘       └──┘          │
│   광대      탑          별           │  ← L4 카드 의미 캡션
│                                     │
│   ─────────────────────             │  ← 로즈골드 구분선
│                                     │
│   [L3] VERDICT                      │
│   ┌─────────────────────────┐       │
│   │   STRONG GO             │       │  ← L4-CTA 판정 배너
│   │   강하게 GO              │       │
│   └─────────────────────────┘       │
│                                     │
│   [L4] 카드는 명확하게 행동을         │
│        추천합니다. 망설이지 마세요.    │
│                                     │
│   [L5] *같은 상황을 반복 추첨하면     │
│        의미가 흐려질 수 있습니다.     │
│                                     │
│   [버튼] 다시 뽑기 / 처음으로          │
│                                     │
└─────────────────────────────────────┘
```

- **전체 정렬**: 완전 중앙 정렬 (Center-aligned)
- **좌우 마진**: 약 **24–32px** (모바일) / 자동 max-width (데스크탑 ~480px)
- **섹션 간 간격**: **40–64px**
- **라벨 → 본문**: **12–16px**
- **본문 → 부가설명**: **8–12px**

## 4. 디자인 원칙 요약

| 원칙 | 적용 방식 |
|------|----------|
| **신비·프리미엄 톤** | 딥 퍼플 + 화이트 + 라벤더 = 결정 보조 도구의 권위감 |
| **명확한 하이어라키** | 5단계 텍스트 위계로 정보 우선순위 명확 |
| **시각적 리듬** | 라벤더 라벨이 반복되며 스캔 포인트 역할 |
| **미니멀리즘** | 장식 최소화, 타이포그래피와 컬러만으로 구조 표현 |
| **세리프 × 산세리프** | 슬로건(세리프)으로 권위감, 본문(산세리프)으로 현대적 가독성 |
| **모바일 우선** | 결정 순간에 휴대폰으로 즉시 켜는 사용 패턴 가정 |

## 6. Tailwind 토큰 매핑

`tailwind.config.ts`에 다음 컬러 토큰을 등록한다:

```ts
theme: {
  extend: {
    colors: {
      'pf-bg': '#2B2042',
      'pf-fg': '#FFFFFF',
      'pf-fg-soft': '#F0EDF5',
      'pf-accent': '#C8A2E8',
      'pf-accent-soft': '#D4B8F0',
      'pf-rose': '#A07060',
      'pf-rose-soft': '#C09080',
      'pf-mute': '#B0AABA',
      'pf-warn': '#D89060',
    },
    fontFamily: {
      'serif-hero': ['"Bodoni Moda"', '"Didot"', 'serif'],
      'sans-kr': ['"Noto Sans KR"', 'sans-serif'],
      'serif-kr': ['"Nanum Myeongjo"', 'serif'],
    },
  },
},
```
